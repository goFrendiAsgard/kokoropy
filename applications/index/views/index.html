Welcome to kokoropy

Kokoropy is a Python web framework that let you make web-application easily and freely

Kokoropy is built with "educational" purpose in mind. This allow you to present your scientific matplotlib figure as easy as never before.

To know more about kokoropy, please visit our getting started and example section.

Kokoropy is also available in github. So that you can download and use it freely
